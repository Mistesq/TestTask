## Getting started guide

1.  Fork repository

2.  Clone repo:

```bash
$ git clone https://github.com/YOUR-USERNAME/TestTask.git
```

3.  Go to project directory and install dependencies

```bash
$ cd TestTask && npm i
```

4.  Run the development server

```bash
$ npm start
```
